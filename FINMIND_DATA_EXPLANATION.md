# 📊 FinMind 資料完整說明

## 🎯 概述

**FinMind** 是一個提供台股原始資料的 API 服務，但 **不提供技術指標和形態識別**。

您的系統架構：
```
FinMind API → 下載原始資料 → 本地計算技術指標 → 形態識別 → 交易訊號
```
