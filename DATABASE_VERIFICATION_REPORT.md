## 4️⃣ ROE 計算邏輯驗證 ✅

### 計算公式
```
ROE = 淨利率 × 資產週轉率 × 權益乘數
```

### 手動驗證
```python
淨利率 = 42.79%
資產週轉率 = 0.49  # (營收 × 4) / 總資產
權益乘數 = 1.53    # 總資產 / 股東權益

ROE = 42.79% × 0.49 × 1.53 = 32.08%
```

### API 回傳
```
ROE = 32.33%
```

### 差異分析
```
API 回傳: 32.33%
手動計算: 32.08%
差異: 0.25% (四捨五入造成)
```

### 年化邏輯驗證
```python
Q3 營收: 759.7B
年化營收 (×4): 3038.8B
總資產: 6165.7B
資產週轉率 = 3038.8B / 6165.7B = 0.4929 ≈ 0.49 ✅
```

**結論**: ✅ 年化邏輯正確，計算準確
