# 📊 多時間週期分析指南

## 🎯 概述

多時間週期分析是技術分析的重要方法，透過觀察不同時間尺度的價格走勢，可以：
- ✅ 提高形態信號的可靠性
- ✅ 確認趨勢的主要方向
- ✅ 找到更好的進出場時機
- ✅ 降低假信號的干擾

---

## 📐 時間週期說明

### 支援的時間週期

| 週期 | 代碼 | 說明 | 建議數據量 | 適用場景 |
|------|------|------|-----------|---------|
| **日線** | D | 每個交易日 | 120天 | 短線交易、當沖 |
| **週線** | W | 每週（週五收盤） | 52週 | 波段操作 |
| **月線** | M | 每月（月底收盤） | 24個月 | 中長期投資 |
| **季線** | Q | 每季（季末收盤） | 12季 | 長期投資 |
| **半年線** | 6M | 每半年 | 8期 | 超長期趨勢 |
| **年線** | Y | 每年（年底收盤） | 5年 | 戰略配置 |

### 移動平均線週期

常用的移動平均線：
- **MA5**: 5日/週/月均線（短期）
- **MA10**: 10日/週/月均線（短期）
- **MA20**: 20日/週/月均線（中期）
- **MA60**: 60日/週/月均線（季線）
- **MA120**: 120日/週/月均線（半年線）
- **MA240**: 240日/週/月均線（年線）

## 📖 分析原理

### 1. 時間週期層級

```
年線 (Y)
  ↓ 確認超長期趨勢
半年線 (6M)
  ↓ 確認長期趨勢
季線 (Q)
  ↓ 確認中長期趨勢
月線 (M)
  ↓ 確認中期趨勢
週線 (W)
  ↓ 確認波段趨勢
日線 (D)
  ↓ 確認短期走勢
```

**基本原則:**
- 大週期決定主要趨勢
- 小週期找進出場點
- 大週期確認小週期的有效性

### 2. 共振原理

**多頭共振範例:**
```
月線: 頭肩底形成 ✅
週線: W底突破 ✅
日線: 回檔整理 ✅
      ↓
   🔥 多頭共振確認
   → 高可信度買進信號
```

**共振強度計算:**
```
共振強度 = 同方向信號數 / 總信號數

評級:
- ≥ 80%: 強勢共振 🔥
- ≥ 60%: 一般共振 ✅
- < 60%: 信號分歧 ⚠️
```

### 3. 操作策略

#### 策略A: 順勢操作（推薦）

```
步驟1: 確認大週期趨勢（月線/週線）
步驟2: 等待小週期回檔（日線）
步驟3: 小週期形態確認後進場
步驟4: 止損設在大週期支撐
步驟5: 目標看大週期目標價
```

**案例: 2330多頭操作**
```
月線: 頭肩底形成，目標2350
週線: W底突破，目標2100
日線: 回檔至1850支撐

操作:
✅ 進場: 日線1850（週線頸線附近）
✅ 止損: 1780（週線頸線）
✅ 目標1: 2100（週線目標）
✅ 目標2: 2350（月線目標）
```

#### 策略B: 逆勢交易（高風險）

```
條件:
1. 小週期（日線）出現反轉信號
2. 大週期（週線/月線）接近關鍵支撐/壓力
3. 成交量配合

風險控管:
⚠️  嚴格止損（2-3%）
⚠️  小倉位測試
⚠️  快進快出
```

## ⚠️ 注意事項

### 1. 數據需求

| 時間週期 | 最少數據量 | 建議數據量 |
|---------|----------|----------|
| 日線 | 60天 | 120-500天 |
| 週線 | 20週 | 52-100週 |
| 月線 | 20個月 | 24-60個月 |
| 季線 | 12季 | 20季 |

**檢查數據:**

```bash
count = db.stock_price.count_documents({'symbol': '2330'})

```

### 2. 週期選擇建議

**短線交易（持倉1-7天）:**
- 主要: 日線
- 輔助: 週線
- 不建議: 月線以上

**波段交易（持倉1-3個月）:**
- 主要: 週線
- 輔助: 日線、月線
- 推薦組合: D + W + M

**長期投資（持倉6個月以上）:**
- 主要: 月線、季線
- 輔助: 週線
- 推薦組合: W + M + Q

### 3. 常見問題

**Q1: 為什麼有些時間週期沒有信號？**

A: 可能原因：
- 數據量不足（需至少20個週期）
- 該時間週期沒有明確形態
- 形態條件不符合嚴格標準

**Q2: 時間週期信號衝突怎麼辦？**

A: 處理方式：
1. 以大週期為主（月線 > 週線 > 日線）
2. 等待信號一致
3. 減小倉位、嚴格止損

**Q3: 共振強度多少才可進場？**

A: 建議標準：
- 強勢共振（≥80%）: 可正常倉位
- 一般共振（60-80%）: 減半倉位
- 信號分歧（<60%）: 觀望或小倉測試
