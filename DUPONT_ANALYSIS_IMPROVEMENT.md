## 📋 改進摘要

針對杜邦分析中「資產週轉率」的判斷邏輯進行改進，加入**產業別自動偵測**與**產業專屬標準**，避免對資本密集產業（如半導體製造業）產生錯誤判斷。

---

## ❌ 改進前的問題

### 問題案例：台積電 (2330)

**原始判斷**：
- 資產週轉率：0.13
- 結論：「資產週轉率過低 (<0.5)，資產閒置嚴重」
- 建議：「處分閒置資產、提升產能利用率、加強營運管理」

### 問題分析

1. **未考慮產業特性**
   - 半導體製造業屬於「重資產」產業
   - 單台 EUV 設備價值約 36 億台幣
   - 正常資產週轉率範圍：0.3 - 0.6

2. **誤判資產閒置**
   - 實際狀況：台積電 CoWoS 產能已被訂滿至 2025 H1
   - NVIDIA 囊括超過 50% 產能
   - 產能利用率約 85-90%（產業正常）

3. **建議不適用**
   - 「處分閒置資產」→ 半導體設備必須持有
   - 應建議「優化產能利用率」、「提升先進製程比例」

## ✅ 改進內容

### 1. 新增產業自動偵測功能

```typescript
/**
 * 偵測產業類型
 */
private detectIndustryType(indicators: {
  netMargin: number;
  assetTurnover: number;
  grossMargin: number;
}): string {
  // 半導體製造業特徵：高毛利率(>40%)、低週轉率(<0.6)、高淨利率(>15%)
  if (indicators.grossMargin > 40 && 
      indicators.assetTurnover < 0.6 && 
      indicators.netMargin > 15) {
    return '半導體製造';
  }
  
  // 傳統製造業特徵：中等毛利率(15-40%)、中等週轉率(0.5-1.5)
  if (indicators.grossMargin >= 15 && 
      indicators.grossMargin <= 40 &&
      indicators.assetTurnover >= 0.5 && 
      indicators.assetTurnover <= 1.5) {
    return '傳統製造';
  }
  
  // 零售業特徵：低毛利率(<30%)、高週轉率(>1.5)
  if (indicators.grossMargin < 30 && 
      indicators.assetTurnover > 1.5) {
    return '零售服務';
  }
  
  // 資本密集產業：低週轉率(<0.5)、中高毛利率(>25%)
  if (indicators.assetTurnover < 0.5 && 
      indicators.grossMargin > 25) {
    return '資本密集';
  }
  
  return '一般產業';
}
```

### 2. 產業別資產週轉率標準

| 產業類型 | 優異 | 良好 | 偏低 | 過低 |
|---------|------|------|------|------|
| **半導體製造** | >0.6 | 0.4-0.6 | 0.3-0.4 | <0.3 |
| **資本密集** | >0.8 | 0.5-0.8 | 0.3-0.5 | <0.3 |
| **傳統製造** | >1.5 | 1.0-1.5 | 0.6-1.0 | <0.6 |
| **零售服務** | >3.0 | 2.0-3.0 | 1.5-2.0 | <1.5 |
| **一般產業** | >1.5 | 0.8-1.5 | 0.5-0.8 | <0.5 |

### 3. 產業專屬建議

#### 半導體製造 / 資本密集產業：
- ✅ 偏低：「優化產能利用率、提升先進製程比例、擴大高附加價值產品」
- ❌ 過低：「檢視產能利用率、評估設備效能、考慮處分老舊產線」

#### 傳統製造 / 零售服務：
- ✅ 偏低：「加速存貨週轉、提高應收帳款回收效率、優化資產配置」
- ❌ 過低：「處分閒置資產、提升營運效率、加強資產管理」
