## 代码示例

### 最小集成示例

```python
from utils.stock_classifier import StockClassifier, SecurityType
from pymongo import MongoClient

# 初始化
client = MongoClient('mongodb://localhost:27017/')
db = client['tw_stock_analysis']
classifier = StockClassifier(db)

# 在下载前判断
def smart_download(stock_id):
    stock_type = classifier.get_type_from_db(stock_id)
    
    if StockClassifier.should_download_financials(stock_type):
        download_financials(stock_id)  # 执行下载
    else:
        print(f"⏭️  跳过 {stock_id}: {stock_type.value}")
```
