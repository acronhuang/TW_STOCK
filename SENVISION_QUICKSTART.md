# SenVision 量化形態選股系統 - 快速使用指南

**版本**: 2.1 (多時間週期分析版)
**更新日期**: 2026-02-24
**新增功能**: ✨ 多時間週期分析（日、周、月、季、半年、年線）

## 🎯 核心特色

### 1. 12神招形態自動識別
- **底部反轉**: W底、破底翻、破底翻W底、頭肩底
- **頂部反轉**: M頭、假突破、頭肩頂、假突破頭肩頂
- **趨勢突破**: 下飄旗形、上飄旗形、收斂三角形

### 2. ✨ 多時間週期分析（NEW!）
- **支援週期**: 日線(D)、週線(W)、月線(M)、季線(Q)、半年線(6M)、年線(Y)
- **共振分析**: 自動識別多個時間週期的一致信號
- **智能建議**: 基於共振強度提供交易建議

### 3. 自動化測幅與風控
- 自動計算目標價（1:1測幅）
- 自動設定停損價
- 計算風報比（Risk/Reward Ratio）

## 🚀 快速開始

### 基礎使用

```bash
# 1. 單一股票多時間週期分析
python3 pattern_recognition/multi_timeframe_analysis.py --symbol 2330

# 2. 指定時間週期
python3 pattern_recognition/multi_timeframe_analysis.py --symbol 2330 --timeframes D W M

# 3. 形態掃描（含交易建議）
python3 pattern_recognition/multi_timeframe_scanner.py --symbol 2330 --timeframes D W M --show-suggestion
```

### 進階使用

```bash
# 1. 週線月線選股（長期投資）
python3 pattern_recognition/multi_timeframe_scanner.py \
  --symbol 2330 \
  --timeframes W M \
  --min-confidence 0.80 \
  --show-suggestion

# 2. 日線週線操作（波段交易）
python3 pattern_recognition/multi_timeframe_scanner.py \
  --symbol 2330 \
  --timeframes D W \
  --min-confidence 0.75 \
  --show-suggestion

# 3. 僅查看基礎分析與均線
python3 pattern_recognition/multi_timeframe_analysis.py \
  --symbol 2330 \
  --timeframes D W M \
  --ma-periods 5 20 60 120
```

## 🎓 使用技巧

### 技巧1: 週線月線選股，日線進場

```bash
# 步驟1: 掃描週線/月線的多頭形態
python3 pattern_recognition/multi_timeframe_scanner.py -s 2330 -t W M

# 步驟2: 確認有共振後，監控日線進場點
python3 pattern_recognition/multi_timeframe_scanner.py -s 2330 -t D --show-suggestion
```

### 技巧2: 大週期確認，小週期操作

**分析流程:**
1. 月線方向 → 確定大方向（多頭/空頭）
2. 週線位置 → 確認波段階段（起漲/中繼/末升）
3. 日線形態 → 尋找進場時機

**範例:**
```
月線: 上升趨勢 ✅ → 大方向做多
週線: W底突破 ✅ → 波段起漲點
日線: 回檔至頸線 → 進場點
```

### 技巧3: 時間週期過濾

**過濾規則:**
```
- 日線信號 → 需要週線確認 → 可信度提升
- 週線信號 → 需要月線確認 → 可信度提升
- 月線信號 → 直接採用 → 最高可信度
```

## 📐 時間週期說明

| 週期 | 代碼 | 說明 | 適用場景 |
|------|------|------|---------|
| **日線** | D | 每個交易日 | 短線交易、當沖 |
| **週線** | W | 每週（週五收盤） | 波段操作 |
| **月線** | M | 每月（月底收盤） | 中長期投資 |
| **季線** | Q | 每季（季末收盤） | 長期投資 |
| **半年線** | 6M | 每半年 | 超長期趨勢 |
| **年線** | Y | 每年（年底收盤） | 戰略配置 |

## ⚠️ 注意事項

### 1. 數據需求

| 時間週期 | 最少數據量 | 建議數據量 |
|---------|----------|----------|
| 日線 | 60天 | 120-500天 |
| 週線 | 20週 | 52-100週 |
| 月線 | 20個月 | 24-60個月 |
| 季線 | 12季 | 20季 |

### 2. 週期選擇建議

**短線交易（持倉1-7天）:**
- 主要: 日線
- 輔助: 週線

**波段交易（持倉1-3個月）:**
- 主要: 週線
- 輔助: 日線、月線
- 推薦組合: D + W + M

**長期投資（持倉6個月以上）:**
- 主要: 月線、季線
- 輔助: 週線
- 推薦組合: W + M + Q

### 3. 共振強度標準

- **強勢共振（≥80%）**: 可正常倉位
- **一般共振（60-80%）**: 減半倉位
- **信號分歧（<60%）**: 觀望或小倉測試

### 操作心法

1. **大週期定方向** - 月線/季線確定趨勢
2. **中週期找位置** - 週線找波段位置
3. **小週期抓時機** - 日線找進出點
4. **嚴守止損線** - 大週期設止損
5. **耐心等共振** - 信號一致再進場
